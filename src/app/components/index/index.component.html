What is YouthDraft?

YouthDraft is a modern tool for an age-old problem. We are bringing the assessment and draft process for youth sports from the clipboard to the digital age.

Still using Excel to manage player assessments? Recover lost time in transferring them to excel using our web and mobile app solution.

YouthDraft is a tool that will save leagues, managers, and coaches countless hours while offering the ability to automate the assessment process.

How can I get YouthDraft?

YouthDraft is available on web and mobile for little league baseball to provide you the easiest way to evaluate talent. Start a league or join an existing one as a coach.

Want it for your sport? Hit the contact button in the bottom right and let us know what sport you want to use YouthDraft in next.
